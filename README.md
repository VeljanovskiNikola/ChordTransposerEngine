# ChordTransposerEngine

A platform-agnostic Swift framework for parsing song text (lyrics interleaved with chord symbols), detecting chords, and transposing them by any number of semitones.

## Targets

- iOS 15+
- macOS 12+
- watchOS 8+
- tvOS 15+
- visionOS 1+

## Usage

```swift
import ChordTransposerEngine
```
